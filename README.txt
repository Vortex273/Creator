Платформа по культуре народов России

Веб-приложение, разработанное на Python, помогает студентам, преподавателям педагогических вузов и школьным учителям быстро находить готовые сценарии мероприятий, посвящённых культуре и традициям народов России.

1. Функциональность

Пользователь выбирает:
Тему: например, "Легенды народов Кавказа", "Праздники Сибири"
Целевую аудиторию: начальная школа, средние классы, старшеклассники
Длительность занятия: (15 минут, 30 минут, целый урок).
Доступный инструментарий: телефоны учеников, компьютер учителя, интерактивная доска

На основе этих параметров приложение предлагает до 5 готовых сценариев, каждый из которых включает:
Название урока.
Целевую аудиторию.
Цель занятия.
Рекомендации по вступлению.
Основную часть (игры, обсуждения, творческие задания).
Раздел рефлексии.
Главный вывод урока.

2. Технологии

Язык программирования: Python 3.14
HTTP-сервер: http.server
Шаблонизация и интерфейс: HTML, CSS, JavaScript
Обработка данных: json

3. Установка и запуск

3.1. Убедитесь, что у вас установлен Python 3.7 или выше
3.2. Щёлкните правой кнопкой мыши внутри папки с проектом. 
В появившемся меню удерживайте клавишу Shift и выберите "Открыть окно PowerShell здесь.
3.3. Запустите сервер, выполнив в терминале команду:
    python server.py
3.4. Откройте браузер и перейдите по адресу: 
    http://localhost:8000
3.5.Приложение готово к использованию!
